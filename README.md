set-styles
==========

Batch-set CSS properties on an element.

## Install

Install `set-styles` from npm or `airportyh/set-styles` from component.

## Usage

```js
var setStyles = require('set-styles')

setStyles(element, {
  position: 'absolute',
  top: 100 + 'px',
  left: 200 + 'px',
  width: 200 + 'px',
  height: 200 + 'px'
})
```
